Types of Lists in HTML

Unordered List (Bulleted List)

An unordered list displays items with bullet points.

  • Apples
  • Oranges
  • Bananas
  • Grapes

Ordered List (Numbered List)

An ordered list displays items with numbers or letters.

  1. First Step
  2. Second Step
  3. Third Step
  4. Fourth Step

Definition List

A definition list is used to display terms and their definitions.

HTML
A markup language used to structure content on the web.
CSS
A stylesheet language used to describe the presentation of a web page.
JavaScript
A programming language used to create interactive effects within web browsers.
